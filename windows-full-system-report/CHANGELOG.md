# Changelog

## 2.0.0 - 2026-07-12

### Preserved
- All original general, hardware, network, port, software, process, service, security, share, startup, task, update and event-log collection areas.

### Added
- Administrator and timing metadata
- Locale, language and NTP information
- Baseboard, chassis, GPU, monitor, audio, battery, USB and PnP error inventory
- Storage reliability, partitions, volumes, BitLocker, shadow copies and TRIM
- Network profiles, advanced properties, counters, DNS settings, neighbors, Wi-Fi, proxy and hosts
- Process names for sockets; TCP state summary
- Store apps, capabilities, PowerShell modules and development runtimes
- Process owners, paths, command lines and service accounts
- Defender preferences and detections, Secure Boot, TPM, Device Guard, UAC, audit policy, certificates, SMB and RDP/WinRM
- Sessions, profiles, SMB sessions, mapped drives and printers
- Startup folders and detailed task actions/triggers
- Windows Update history, DISM inventory, component-store status and reboot indicators
- Hyper-V, WSL, Docker and PowerShell policy checks
- Reliability history, boot events, bugchecks, SFC and DISM checks
- Per-block execution summary
